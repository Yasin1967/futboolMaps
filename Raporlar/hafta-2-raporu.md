# 🏗️ FootballMaps Türkiye - Hafta 2 Raporu

## 📅 Yapılan Çalışmalar
Bu hafta veritabanı bağlantısı ve ORM yapılandırması üzerinde yoğunlaşılmıştır.

### ✅ Tamamlanan Maddeler
- **FootballMapsDbContext:** Veritabanı yönetim katmanı EF Core ile yapılandırıldı.
- **Entity Framework Core:** SQL Server Express LocalDB bağlantı dizesi `appsettings.json` dosyasına eklendi.
- **Migrations:** Veritabanı şeması oluşturmak için `InitialCreate` migration'ı hazırlandı.
- **Veritabanı Oluşturma:** Şema başarıyla fiziksel veritabanına uygulandı.
